# Pathfinder Horology
> One-of-One Masterpieces

A premium, luxury e-commerce front-end showcase mimicking the aesthetics of Swiss watchmakers like Rolex. Built with React, Vite, and GSAP/Framer Motion to deliver cinematic scroll-based animations, precision typography, and flawless layout transitions.

## Features
- **Cinematic Scroll Rendering**: Uses `gsap` ScrollTrigger paired with an HTML5 `<canvas>` to deliver a high-performance, 240-frame sequence of a watch assembling in 3D right as you scroll down the page.
- **Micro-Animations & Micro-Interactions**: Features elegant, staggered reveals and continuous hovering `framer-motion` sequences over pristine, AI-extracted macro product images.
- **Swiss Precision Typography**: Hand-tuned leading and tracking on classic serif fonts (`Cormorant Garamond`) contrasting heavily with functional sans-serif elements. 
- **By Invitation Only Paradigm**: Sophisticated CSS grid flexbox layouts delivering an exclusive contact point design format, stripped of standard forms.

## Tech Stack
- Frontend Framework: **React 18**
- Build Tool: **Vite**
- Animation Engines: **GSAP** & **Framer Motion**
- Styling: **Pure CSS** (Custom Variables & Modern Native Flexbox/CSS Grid)
- Icons: **Lucide React**

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the local dev server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

## Design Philosophy

The aesthetic objective is strictly oriented toward massive luxury appeal—dominating negative space with immense graphics without sacrificing vertical pixel rhythm. The transition from the pitch-black 3D hero assembly reel directly to the pristine "platinum" white gallery forces a stunning visual tempo change for the user, drawing complete attention to the product.
